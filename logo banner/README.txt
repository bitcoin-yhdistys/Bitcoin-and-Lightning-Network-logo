# Bitcoin Banner PowerPoint -template

>Editable svg:   bitcoin-banner text-template-font.svg
>Needs Font:     Ubuntu Bold Italic

## Create Bitcoin Banner with custom text
-Edit  “banner template text” on next page
-Select all (CTRL+A)
-R-click
-Save As Picture
-Select Save as Type: "Scalable Vector Graphics (*.svg)"
-Edit SVG with your favorite vector graphics editor
Vóila!
